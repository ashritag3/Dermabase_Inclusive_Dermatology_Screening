## ğŸ§  How It Works (Technical Overview)

Dermabase uses a **Convolutional Neural Network (CNN)** with two main stages:

### 1. Feature Extraction
- **Convolution layers** scan small sections of an image to detect patterns
- **Pooling layers** reduce noise and adjust for:
  - Skin tone  
  - Lighting  
  - Shadows  

### 2. Classification
- Extracted features are passed into a neural network
- Uses **supervised learning** to predict conditions based on labeled data
- **Backpropagation** allows the model to learn from errors and improve over time

As more users contribute scans, the model becomes **more accurate and more inclusive**.
